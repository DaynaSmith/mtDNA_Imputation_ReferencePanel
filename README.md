**Reference Panel for imputing untyped mitochondrial SNPs**

The reference panel included in this repository is an updated version of the panel created by McInerney et al. (2021).
This reference panel includes the 36,960 mitochondrial sequences from the original reference alignment (```McInerney_Master_Alignment_July18_2018.fasta.gz```) 
in addition to 21,422 new mitochondrial sequences that have been added to GenBank since 18 July 2018.
